🚀 INICIO RÁPIDO (3 pasos | 11 minutos)
════════════════════════════════════════════════════════════════════════════

1️⃣ INSTALAR (5 minutos)
   
   PowerShell (como administrador):
   
   cd c:\xampp\pdt
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   pip install -r requirements.txt


2️⃣ CONFIGURAR SUPABASE (5 minutos)
   
   a) Ir a https://supabase.com
   b) Crear cuenta gratuita
   c) Crear proyecto
   d) Settings > API
      - Copiar Project URL → SUPABASE_URL en .env
      - Copiar anon key → SUPABASE_KEY en .env
   e) Settings > Database
      - Copiar contraseña → DATABASE_URL en .env


3️⃣ EJECUTAR (1 minuto)
   
   python main.py
   
   Verás:
   INFO:     Uvicorn running on http://127.0.0.1:8000

💾 BASE DE DATOS (Automática)
════════════════════════════════════════════════════════════════════════════

Se crean 4 tablas PostgreSQL:

PACIENTES (30 campos)
  ├── Datos personales (cedula, nombre, apellido, email, teléfono)
  ├── Datos médicos (alergias, medicamentos, historial)
  ├── Tutor info (nombre, teléfono, email)
  └── Control (activo, created_at, updated_at)

CONSULTAS (20 campos)
  ├── Vitales (peso, altura, IMC auto-calculado, temperatura, etc)
  ├── Diagnóstico (motivo, diagnóstico, medicinas, observaciones)
  ├── Pago (monto, estado_pago)
  └── Timestamps (fecha_consulta, próxima_cita, created_at)

GASTOS (11 campos)
  ├── Descripción, monto, categoría
  ├── Recurrencia (recurrente, frecuencia)
  └── Timestamps

INGRESOS (10 campos)
  ├── Descripción, monto, tipo, método_pago
  ├── Referencias (paciente_id, consulta_id opcionales)
  └── Timestamps

📊 EJEMPLOS DE USO
════════════════════════════════════════════════════════════════════════════

Crear paciente (PowerShell):
  $body = @{
    cedula = "123456789"
    nombre = "Juan"
    apellido = "García"
  } | ConvertTo-Json
  
  Invoke-WebRequest -Uri "http://localhost:8000/api/pacientes" `
    -Method POST -Body $body `
    -Headers @{"Content-Type"="application/json"}


Crear consulta:

  {
    "paciente_id": 1,
    "peso": 35,

🔥 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

Error: "ModuleNotFoundError: No module named 'fastapi'"
  → pip install -r requirements.txt

Error: "Connection refused" (Supabase)
  → Verifica DATABASE_URL en .env
  → Verifica Supabase está online

Error: "Port 8000 already in use"
  → Cambia port en main.py (último parámetro)
  → O mata el proceso: Get-Process -Name python | Stop-Process

Error: "Venv no activa"
  → Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
  → .\venv\Scripts\Activate.ps1

Error: "Python not found"
  → Descarga de python.org
  → Marca "Add Python to PATH" durante instalación
  → Reinicia PowerShell
