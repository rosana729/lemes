# 🚀 Checklist Rápido - Mostrar Proyecto en GitHub Pages

## ⚡ 3 Pasos Principales

### **Paso 1: Habilitar GitHub Pages (2 minutos)**

```
1. Ve a: https://github.com/rosana729/lemes/settings/pages
2. En "Source": Selecciona branch "main" y folder "/docs"
3. Click "Save"
4. Espera 1-2 minutos ✅
```

**Tu sitio estará en**: `https://rosana729.github.io/lemes`

---

### **Paso 2: Crear GIF Demostrativo (10 minutos)** - OPCIONAL pero recomendado

```
1. Descarga ScreenToGif: https://www.screentogif.com/
2. Ejecuta tu servidor: python main.py
3. Abre: http://localhost:8000/docs
4. Graba un GIF (15 segundos max) mostrando los endpoints
5. Guarda como docs/demo.gif
6. Commit y push
```

**Guía completa**: [GIF_DEMO.md](GIF_DEMO.md)

---

### **Paso 3: Hacer Push de Cambios**

```bash
git add .
git commit -m "Add GitHub Pages site and documentation"
git push origin main
```
